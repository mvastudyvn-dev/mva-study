
# MVA Study - Design System

## Mục tiêu
Thiết kế website E-learning hiện đại, tối giản, tông màu cam, thân thiện với học sinh.

## Phong cách
- Modern SaaS Dashboard
- Nhiều khoảng trắng
- Bo góc 16–24px
- Shadow mềm
- Responsive Desktop/Tablet/Mobile

## Responsive

Tablet 768-1199
Mobile <768

## Quy tắc
- Khoảng trắng rộng.
- Không quá 3 màu chính.
- Card đồng nhất.
- Nút CTA luôn màu cam.
- Dashboard ưu tiên dễ đọc.
